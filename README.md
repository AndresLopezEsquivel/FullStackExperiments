# Web Development

A **Full Stack Experiments** repository — a personal sandbox where I build small,
self-contained projects to learn and practice web development. Each subdirectory
is an independent experiment, ranging from static HTML/CSS layouts to
Node.js-powered front-end/back-end applications.

## Projects

| Project | Type | Description |
| --- | --- | --- |
| [BusinessCard](BusinessCard) | Front end | A static business card / profile page. |
| [Calculator](Calculator) | Front end | A simple calculator experiment. |
| [CityCard](CityCard) | Front end | A city information card layout. |
| [CoworkingSpaceSite](CoworkingSpaceSite) | Front end | A landing page for a coworking space. |
| [DummyServer](DummyServer) | Back end | A test Node.js server for trying out APIs and data handling. |
| [Google](Google) | Front end | A clone of the Google search homepage. |
| [I_Robot](I_Robot) | Front end | A themed page experiment. |
| [PostsSite](PostsSite) | Full stack | A posts app with a separate Node.js API server and front-end server. |
| [ProductSplash](ProductSplash) | Front end | A product splash / marketing landing page. |
| [SciStream](SciStream) | Front end | A streaming-style site layout. |

## Tech Stack

- **Front end:** HTML, CSS, and vanilla JavaScript.
- **Back end:** Node.js (ES modules), with `nodemon` for development.

## Running a Project

Static projects can be opened directly in a browser by opening their
`index.html`. For the Node.js projects (e.g. `DummyServer`, `PostsSite`), install
dependencies and start the server:

```bash
cd <project>
npm install
npm start    # or: npm run dev
```

## Purpose

This repository is intentionally a collection of experiments rather than a single
product. It's where I try out new ideas, patterns, and tools across the full
stack.
